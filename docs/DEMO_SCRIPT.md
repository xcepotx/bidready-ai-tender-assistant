# BidReady AI Demo Script

## Goal

Show how BidReady AI helps a bid team move from tender/RFP document review into executive-ready bid/no-bid decision, response planning, evidence tracking, and exportable proposal artifacts.

## Demo Flow

1. Open BidReady AI dashboard.
2. Select a tender project.
3. Set Output Language to Indonesia or English.
4. Click Regenerate All Artifacts.
5. Review Executive Dashboard.
6. Review Requirements.
7. Review Clarifications.
8. Review Response Plan.
9. Review Proposal Outline.
10. Review Evidence Pack.
11. Review Decision Gate.
12. Export Excel Readiness Matrix.
13. Export DOCX Proposal Draft.

## Key Selling Points

- Reduces manual tender review effort.
- Turns tender documents into structured requirements.
- Creates clarification questions automatically.
- Builds response plan and proposal outline.
- Tracks evidence gaps before submission.
- Gives owner-level bid/no-bid decision gate.
- Supports English and Bahasa Indonesia outputs.
- Produces Excel and DOCX artifacts for review.

## Closing Message

BidReady AI helps bid teams review tenders faster, identify risks earlier, prepare proposal artifacts, and make better bid/no-bid decisions.
